# Simple Budgeting Software for Elderly Users

## Overview
This project is a Python-based budgeting application designed to help elderly users manage their finances in a simple and easy way.

## Problem
Many budgeting apps are too complex for older adults. They often include confusing layouts, small text, and too many features, making them difficult to use.

## Solution
This application will allow users to:
- enter monthly income
- track expenses such as rent, groceries, utilities, and medication
- automatically calculate remaining balance
- view simple summaries of spending

## Target Users
Elderly individuals, especially those living on fixed incomes.
