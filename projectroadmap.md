# Project Roadmap

## Project Title
Simple Budgeting Software for Elderly Users

## Sprint 1 Work

### Code Base Reviewed
I reviewed a Python budgeting/expense tracking code base to understand how budgeting applications are structured.

### What I Learned
- Budget applications track income and expenses
- They calculate totals automatically
- They use categories like food, rent, and bills
- Simplicity is important for usability

### Evaluation
The code base was useful but not designed for elderly users. It lacked simplicity and accessibility.

### Attempt to Run
